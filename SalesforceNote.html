1. To enable user to has Console app in App menu, tick the option “Service Cloud User” in user profile


2. Custom Setting (List) will be replaced with Custom Metadata Types (Query with SOQL, end with "__mdt")


3. By setting the “Login IP Ranges” in Profile can skip login verification code


4. To add VF page to the layout of Console app, set in SObject’s layout setting, menu link “Custom Console Components”
